Attention: n'utilisez pas cette extension avec Tab Groups (Panorama) Cette extension met en œuvre le concept d'«  onglets sous la forme de structure arborescente  ». Elle cache les onglets par défaut (en haut) et réduit (facultativement) les bordures épaisses autour de la barre de navigation. À la place, le nouvel arbre d'onglets apparaît sur la gauche (ou la droite). Cette extension a été conçue comme alternative à Tree Style Tab afin de fournir de meilleures performances, stabilité et facilité d'utilisation. Si vous n'avez jamais utilisé d'«  onglets sous la forme de structure arborescente  », que vous ne savez pas à quoi ça peut servir et que vous avez de nombreux onglets ouverts (pas juste 3 ou 4) alors essayez-le. C'est une manière très pratique de gérer les onglets quand vous en avez un grand nombre ; certaines personnes (dont moi) ne peuvent pas imaginer leur navigateur principal avec les onglets ordinaires en haut. Raccourcis clavier (pas encore configurable):
  • F8 — Basculer une (ou deux) des quatre options de masquage automatique du panneau d'onglets selon l'état courant de la fenêtre (maximisé/normal/plein écran/un seul onglet):
    • Quand une fenêtre active est en plein écran, F8 bascule l'option pour les fenêtres en plein écran
    • Quand une fenêtre active est maximisée, F8 bascule l'option pour les fenêtres maximisées
    • Quand une fenêtre active est normale, F8 bascule l'option pour les fenêtres normales
    • Quand une fenêtre active n'a qu'un seul onglet, F8 bascule l'option pour les fenêtres n'ayant qu'un seul onglet
  • Ctrl+Maj+Pg.Pré/Pg.Suiv — Déplace l'onglet actif (avec ses fils) vers le haut/bas
  • Ctrl+Alt+Maj+Pg.Pré/Pg.Suiv — Déplace lentement l'onglet actif (avec ses fils) vers le haut/bas, c.-à-d. en entrant dans un sous-arbre quand c'est possible (attention: les distributions GNU/Linux basées sur KDE utilisent par défaut ces raccourcis pour 'Redémarrer sans confirmation' / 'Éteindre sans confirmation')
  • Maj+Alt+Pg.Pré/Pg.Suiv — Déplace rapidement l'onglet actif (avec ses fils) vers le haut/bas, c.-à-d. sans entrer dans les sous-arbres
  • Ctrl+Maj+,/. — Diminue/augmente l'indentation de l'onglet actif d'un cran (ne fonctionne pas avec les claviers français « azerty » ou « bépo »)
  • Ctrl+Alt+Maj+F — Sélectionne la barre de recherche d'onglets
  • Maj+Double-clic sur le panneau d'onglets — Ouvre une nouvelle fenêtre
  • Glisser-déposer un lien depuis une page web sur l'arbre (facultativement en maintenant Maj) — Ouvre un lien dans un nouvel onglet de premier/arrière plan (selon le paramètre de Firefox « Général -> Lors de l’ouverture d’un lien dans un nouvel onglet, basculer vers celui-ci immédiatement (browser.tabs.loadInBackground) », maintenir Maj inverse ce comportement)
Si vous avez une puce graphique Intel et un problème étrange quelconque avec cette extension (p. ex. Tab Tree n'est pas visible), veuillez essayer de désactiver « Utiliser l’accélération graphique matérielle si disponible » dans les paramètres de Firefox (Préférences -> Avancé -> Général). Pour rapidement désactiver cette extension, utilisez Ctrl+Maj+A pour ouvrir le Gestionnaire de modules complémentaires.
